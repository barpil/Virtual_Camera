# Virtual Camera

### Running in terminal 
```
-f ./filename.txt   -> path to figure file to render (required)
-m moveSpeed        -> camera movement speed (default 3)
-r rotationSpeed    -> camera rotation speed (default 30)
-h                  -> help
```



### Controls
```
WSAD LShift Space - camera movement
LMouse or Arrows  - camera rotation XY axes
QE                - camera roll
R                 - leveling to closest horizon of camera (Z axis leveling)
P                 - take photo and save in ./ folder
MouseScroll       - camera zoom
```
